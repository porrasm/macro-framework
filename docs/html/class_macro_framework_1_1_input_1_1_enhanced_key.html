bool  extended 
)
inlinestatic

Get the KKey equivalent of this VKey. If an extended key is not found, a non-extended version is returned if possible.

Parameters
keyThe VKey to cast into a ScanCode
extendedSet true to prioritize the extended version of the key (Example: Enter vs NumpadEnter). Returns non-extended version as fallback if not found and vice versa.
