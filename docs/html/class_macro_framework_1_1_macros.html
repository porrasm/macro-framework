◆ PauseAndStopHooks()

static void MacroFramework.Macros.PauseAndStopHooks ( )
inlinestatic

This effectively pauses the entire application by disabling input hooks and all Command and MacroFramework.Commands.CommandActivator instances. Use this method if you wish to pause all functionality. Only the MainLoopCallback and async methods which started before tha pause will run.

◆ SetPause()

static void MacroFramework.Macros.SetPause ( bool  paused)
inlinestatic

Setting Paused to true will disable all macro functionality. The hooks will stay on and the MainLoopCallback and MacroFramework.Input.InputEvents.InputCallback callbacks are still called as well as any async methods which which were started before tha pause.
