Enumerator
NotRunning 

The application is not running

Running 

The application is running in a normal state

RunningInLimitedMode 

The application is running in limited mode in which only certain IActivator are updated. Input is disabled.

Paused 

The application is paused and no Command or IActivator instances are automatically updated
