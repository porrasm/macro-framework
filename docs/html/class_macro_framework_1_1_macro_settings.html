◆ KeyStateFixTimestep

int MacroFramework.MacroSettings.KeyStateFixTimestep = 5000

The Input.KeyStates can lose track of keystates if another application intercepts keys. This can cause keys to get stuck down and Binds stop working. Use this timestep to reset the keys using Input.KeyStates.ResetKeyStates at regular intervals if no keys have been pressed for a while. Set to 0 to disable automatic keystate resets.
